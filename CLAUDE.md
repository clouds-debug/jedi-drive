# Jedi Drive — контекст проекта

Автошкола в Тбилиси, основатель Анри (23 года). Запущена в 2023 (онлайн-теория), своя площадка с 2025. Сайт собирается итеративно с пользователем — это его проект, я помогаю.

- **Next.js 16.2.9** App Router (читать доки в `node_modules/next/dist/docs/` перед изменениями — версия отличается от обычной)
- **React 19.2.4**
- **Tailwind v4** (конфиг через `@theme` в `app/globals.css`, нет `tailwind.config.js`)
- **TypeScript 5** (Vercel строгая сборка — следить за unused vars, `@ts-expect-error` ловит ошибки)
- **Manrope** (cyrillic) для основного текста, Geist Mono для моноширинного
- **Хостинг:** Vercel (auto-deploy с GitHub `main`)
- **Repo:** https://github.com/clouds-debug/jedi-drive
- **Live:** https://jedi-drive.vercel.app

## Бренд и палитра

Это критично — пользователь несколько раз возвращался к палитре.

- **Основной:** navy `#141A3D` — фоны секций, шапка, всё тело сайта (html/body фон тоже navy)
- **Глубокий navy:** `#0F1430` для футера и тёмных вариантов
- **Оранжевый акцент:** `#F97316` — только акценты, CTA, бордеры, метки. Никогда не доминирует.
- **Оранжевый мягкий:** `#FDBA74` для тонких подписей
- **Текст на navy:** `#A5B4D8` (muted-on-navy)

Все секции на едином `#141A3D` без ступенек по цвету. Между секциями — тонкая оранжевая нить градиентом сверху: `<div className="absolute inset-x-0 top-0 h-px bg-gradient-to-r from-transparent via-orange/30 to-transparent" />`.

Кремового нет, белого как фона нет. Светлые секции с homepage были убраны по требованию пользователя.

## Design system

### Карточки

Стандарт: `bg-white/[0.03] border border-white/10 border-l-[3px] border-l-orange rounded-[var(--radius-card)]` + glow blob в углу:
```tsx
<div className="absolute -right-12 -top-12 w-44 h-44 bg-orange/[0.10] rounded-full blur-[60px] pointer-events-none" />
```
Hover: `hover:bg-white/[0.05] hover:border-white/20 hover:-translate-y-1`.

Highlighted/Premium вариант: border `border-orange/40 border-l-[3px] border-l-orange` + усиленный glow `bg-orange/[0.18]`.

### Лейблы секций

`<SectionLabel num="01">Подход</SectionLabel>` — формат `01 ──── Подход`:
- Моноширинный оранжевый номер
- Тонкая градиентная линия `bg-gradient-to-r from-orange/55 to-transparent`
- Название в смешанном регистре (НЕ UPPERCASE)

Никогда не использовать «ПОЧЕМУ МЫ» / «ОТЗЫВЫ» капсом — пользователь это раскритиковал как «не элитарно».

### Reveal анимация

`<Reveal delay={i * 60}>` для появления при скролле через IntersectionObserver. В `components/Reveal.tsx`. Принимает `delay, y, className`. НЕ принимает `as` prop (был баг с TS).

### Hero-rise

CSS keyframe `hero-rise` для появления первого экрана. Использовать с inline `style={{ animationDelay: '80ms' }}` для stagger.

### Заголовки секций

Шаблон: «Что мы делаем **по-другому**» — белый текст + одно ключевое слово оранжевым через `<span className="text-orange">…</span>`. 28-34px, weight 500, tracking `-0.015em`, max-width `540px`.

## Контент-решения

- **Категория только B** (не A, никаких других)
- **Языки** ru и ge (не ka), пока сделан только ru, i18n отложен
- **Никаких ложных обещаний:** «документы мы НЕ оформляем», «если не сдал — НЕ учим бесплатно». Это уже исправлялось — следить.
- **Главная без цен** — цены только на страницах услуг
- **Прайсинг практики:** 3 пакета — Только площадка (₾500/10), Площадка + город (₾1100/20), Только город (₾650/10). По 1 уроку ₾70.
- **Прайсинг теории:** В группе ₾450, индивидуально ₾900 (за 16 занятий)

## Инструкторы

19 инструкторов в `lib/instructors/data.ts`. Каждый: id, name, initials, avatarColor (indigo/orange/violet/emerald/rose), rating, reviewsCount, experienceYears, car, languages, bio, reviews[].

В Nav.tsx «Инструкторы» подсвечена активной через `usePathname`.

## Билеты

Сейчас 20 демо-вопросов категории B в `lib/tickets/data.ts`. **Не реальная база** — пользователь сказал что выкупит лицензию у teoria.on.ge.

Поведение квиза (важно — спорил с пользователем):

- **Объяснения показываются только на экране разбора в конце**, не во время теста
- Это поведение «как на экзамене» — пользователь явно просил

Поддержка картинок в `Question.image` (URL). Сейчас 3 демо-SVG в `public/tickets/`.

## Атмосфера главной (HomepageAmbient)

Пользователь хотел крупные тематические декорации (не мелкие). Сейчас:
- **Дороги по обоим краям** (`Road` component) — асфальт-градиент с белыми обочинами и **анимированной оранжевой разметкой** (`road-dash` keyframe, dashed line scrolls)
- **Спидометр** 180px справа со стрелкой (`speedo-needle` keyframe, swing 9s ease-in-out)
- **Dashboard индикаторы** слева (GE active, ABS/ESP/AKB dim)
- **Engine** + индикатор топлива внизу слева
- **● LIVE** pulse справа сверху
- **Вертикальные подписи**: «Jedi Drive · 2026» слева, «Tbilisi · 41.69°N · 44.80°E» справа
- **Glow blob** оранжевый в центре сверху

Видно только при `xl+` (1280+). Все анимации честно гасятся при `prefers-reduced-motion: reduce`.
